## React-Music-Player


### 一、简介
基于React实现的音乐播放器，技术栈 react + react-router + webpack + sass + es6；基本动图见下面：

![首页](/index.png)

### 二、如何执行

####  1、将项目克隆到本地
```javascript
git clone git@github.com:kovlento/music_player_react.git
```
#### 2、安装依赖
```javascript
npm install
```
#### 3、执行
```javascript
npm start
// npm run build(打包)
```
#### 4、打开浏览器浏览 http://localhost:10086/
